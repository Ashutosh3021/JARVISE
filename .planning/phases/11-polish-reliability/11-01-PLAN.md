<task type="auto">
  <name>Task 3: Build project_detector.py</name>
  <files>context/project_detector.py</files>
  <action>
Create project_detector.py with:
- ProjectInfo dataclass: language, framework, test_command, build_command, project_root, has_git
- detect_project(path): Detect project type by scanning for:
  - *.py -> Python (pytest, python -m pytest)
  - package.json -> Node.js (npm test, npm run)
  - Cargo.toml -> Rust (cargo test)
  - go.mod -> Go (go test)
  - requirements.txt / pyproject.toml -> Python
  - Makefile -> C/C++/generic
  - .csproj -> C# (.NET)
- get_test_command(project_path): Return appropriate test command
- get_project_root(): Find project root by looking for markers (git, package.json, etc.)
- has_git_repo(): Check if current directory is in a git repo
- get_git_status(): Get branch name and modified files
  </action>
  <verify>python -c "from context.project_detector import detect_project; p = detect_project('.'); print(f'Language: {p.language}, Test: {p.test_command}')"</verify>
  <done>ProjectInfo with language, framework, test/build commands</done>
</task>
