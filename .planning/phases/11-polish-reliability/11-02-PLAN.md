  <action>
Implement CommandRouter.route():
- Input: user_input string
- Process:
  1. Normalize input (lowercase, strip whitespace)
  2. Check exact matches in _direct_commands
  3. Check fuzzy matches (Levenshtein distance < 3)
  4. Check wildcard patterns
  5. Check learned commands from preference memory
- Output: RouterResult with:
  - DIRECT_TOOL: tool_name, tool_args, confidence=1.0
  - UNKNOWN: confidence < 0.5 -> route to LLM
  - LLM_AGENT: explicitly requested or high confidence it's a question
- Log routing decision for debugging
  </action>
